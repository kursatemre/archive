## ğŸš€ Getting Started

### Option 1: Use Online
Visit [Archive](https://archive.webusta.org) and start previewing ZIP files immediately.

### Option 2: Self-Host

```bash
# Clone the repository
git clone https://github.com/EXELANCE-LLC/Archive.git

# Navigate to the project
cd Archive

# Serve the public folder with any static server
npx serve public

# Or use Python
python -m http.server 8000 --directory public
```

## ğŸ” Privacy & Security
